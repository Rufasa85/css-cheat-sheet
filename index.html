Css Tip & Tricks

These are a few of my favorite CSS snippets!

Salmon Background

Set your apps background to salmon so you can see that your css is linked!

body{
    background:salmon;
}

Flex Center

Get your items to show up right in the middle with flexbox centering

.container{
    display:flex;
    justify-content:center;
    align-items:center;
}

Glowy shadow

Get the cool shadow effect of these cards!

.card:hover, .card:hover h3 {
    box-shadow: 0 0 10px inset deepskyblue,
    0 0 10px deepskyblue;
}

Two-toned background

Sweet two toned affect for a 90s feel

body {
    background:linear-gradient(
    120deg, 
    salmon 0%,
    salmon 50%,
    cornflowerblue 50%)
}
